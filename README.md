
# Game ROM Loader

Binary Ninja loader for some game ROMs.

## features

+ [x] load GBA (GameBoy Advance) ROMs (`armv4`)
+ [x] load NDS (Nintendo DS) ROMs (`armv4`/`armv5`)
+ [ ] planned: PSP (PlayStation Portable) EBOOT (`mips`)
+ [ ] planned: Wii (Nintendo Wii) binary (`powerpc`)
